# ys-portfolio

Personal portfolio site built with React + Vite.

## Tech Stack

- React
- Vite
- React Router
- lucide-react

## Features

- Light / Dark mode toggle
- Responsive layout
- Project showcase with GitHub / demo links
- Separate projects listing page (`/projects`)
